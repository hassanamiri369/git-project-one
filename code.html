اگر شما نیاز به ارسال تغییرات یا push کردن دارید باید اطلاعات نام کاربری و ایمیل اکانت GitHub خود را به کمک دستورات زیر وارد نمایید :

git config –global user.name “Mohsen Derambakht”

git config global user.email “derambakht@gmail.com”

برای مشاهده ی لیست ریموت ریپازیتوری های یک پروژه در گیت میتوانید از دستور زیر استفاده کنید :

git remote -v
